# ReNoobChain In Python
source - [ReNoobChain](https://github.com/isaurssaurav/reNoobChain)

## Setup and Usage

```
$ git clone https://github.com/isaurssaurav/reNoobChainInPython.git

$ cd reNoobChainInPython/

$ python3 -m venv venv

$ . venv/bin/activate

$ pip install -r requirements.txt 

$ python3 main.py
```
